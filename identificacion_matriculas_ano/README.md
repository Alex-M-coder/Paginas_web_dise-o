# 🚗 Matrículas 360 - Identificador de Matrículas de España

Matrículas 360 es una aplicación web interactiva diseñada para calcular de forma inmediata el año de matriculación y la información del tipo de vehículo a partir de cualquier matrícula de España.

## ✨ Características Principales

*   **Soporte Histórico:** Detecta matrículas del sistema moderno de tres letras (2000-actualidad), del sistema provincial alfanumérico (1971-2000) y del sistema numérico provincial clásico (1900-1971).
*   **Diferenciación de Vehículos:** Soporte específico para automóviles, motocicletas, ciclomotores, vehículos históricos y remolques.
*   **Localización (Multi-idioma):** Interfaz totalmente traducida a cinco idiomas: Español (ES), Inglés (EN), Catalán (CA), Gallego (GL) y Euskera (EU).
*   **Estética Visual Premium:** Animación de orbes brillantes en el fondo, transiciones fluidas y una presentación de resultados clara y moderna.
*   **Responsive:** Funciona en ordenadores, teléfonos inteligentes y tabletas sin perder calidad visual.

## 📂 Archivos del Proyecto

*   `index.html`: Estructura HTML del identificador, contenedores de resultados y selector de idiomas.
*   `style.css`: Hojas de estilo con variables de color dinámicas, animaciones y diseño adaptativo.
*   `app.js`: Motor JavaScript para la validación y cálculo de años históricos según la combinación de letras/números, además del sistema de traducción de interfaz.

## 🚀 Cómo Ejecutar

Para abrir y probar la aplicación:
1. Abre el archivo `index.html` en tu navegador favorito.
2. Opcionalmente, puedes utilizar una extensión como **Live Server** en tu editor de código para contar con recarga en tiempo real.
