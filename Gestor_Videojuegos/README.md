# 🎮 GameVault - Gestor de Videojuegos Multiplataforma

GameVault es una aplicación web interactiva diseñada para registrar, organizar y gestionar tu biblioteca personal de videojuegos de múltiples plataformas en un solo lugar.

## ✨ Características Principales

*   **Diseño Premium:** Interfaz oscura y futurista con efectos de iluminación dinámica (*glowing orbs*) y bordes de cristal (*glassmorphism*).
*   **Gestión Dinámica:** Añade nuevos títulos a tu catálogo, edita información existente, filtra por plataforma o estado, y elimina videojuegos.
*   **Decisión Aleatoria (¿A qué jugar hoy?):** Si tienes dudas sobre qué jugar, la aplicación incluye una función para elegir un videojuego al azar de tu lista.
*   **Responsive Design:** Optimizado para ordenadores, tabletas y dispositivos móviles.

## 📂 Archivos del Proyecto

*   `index.html`: Define la estructura semántica de la interfaz de usuario, modales y formularios.
*   `style.css`: Contiene los estilos visuales, efectos de animación avanzados y diseño adaptivo.
*   `app.js`: Implementa la lógica del gestor, almacenamiento local (*LocalStorage* si está disponible) y manipulación interactiva del DOM.

## 🚀 Cómo Ejecutar

Para abrir y probar la aplicación:
1. Abre el archivo `index.html` en cualquier navegador web moderno.
2. Opcionalmente, puedes utilizar una extensión como **Live Server** en tu editor de código para una experiencia óptima con recarga automática.
